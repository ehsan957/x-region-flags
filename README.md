Display Country Flags of X (Twitter) Users

This script is written for the Tampermonkey extension (tested on Firefox) and displays the country flag of users next to their names on X (Twitter).

Note:
If a user is connected through a VPN, the detected country may be incorrect.

How to Use

Install the Tampermonkey extension from the link below.
